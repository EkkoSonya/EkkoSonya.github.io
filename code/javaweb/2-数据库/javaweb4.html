                                    SQL иҜӯеҸҘ

                                    з»“жһ„еҢ–жҹҘиҜўиҜӯиЁҖпјҲStructured Query Languageпјүз®Җз§°SQLпјҢиҝҷжҳҜдёҖз§Қзү№ж®Ҡзҡ„иҜӯиЁҖпјҢе®ғдё“й—Ёз”ЁдәҺж•°жҚ®еә“зҡ„ж“ҚдҪңгҖӮжҜҸдёҖз§Қж•°жҚ®еә“йғҪж”ҜжҢҒSQLпјҢдҪҶжҳҜд»–д»¬д№Ӣй—ҙдјҡеӯҳеңЁдёҖдәӣз»Ҷеҫ®зҡ„е·®ејӮпјҢеӣ жӯӨдёҚеҗҢзҡ„ж•°жҚ®еә“йғҪеӯҳеңЁиҮӘе·ұзҡ„вҖңж–№иЁҖвҖқгҖӮ

                                    SQLиҜӯеҸҘдёҚеҢәеҲҶеӨ§е°ҸеҶҷпјҲе…ій”®еӯ—жҺЁиҚҗдҪҝз”ЁеӨ§еҶҷпјүпјҢе®ғж”ҜжҢҒеӨҡиЎҢпјҢе№¶дё”йңҖиҰҒдҪҝз”Ё;иҝӣиЎҢз»“е°ҫпјҒ

                                    SQLд№ҹж”ҜжҢҒжіЁйҮҠпјҢйҖҡиҝҮдҪҝз”Ё--жҲ–жҳҜ#жқҘзј–еҶҷжіЁйҮҠеҶ…е®№пјҢд№ҹеҸҜд»ҘдҪҝз”Ё/*жқҘиҝӣиЎҢеӨҡиЎҢжіЁйҮҠгҖӮ

                                    жҲ‘д»¬иҰҒеӯҰд№ зҡ„е°ұжҳҜд»ҘдёӢеӣӣз§Қзұ»еһӢзҡ„SQLиҜӯиЁҖпјҡ

                                    • ж•°жҚ®жҹҘиҜўиҜӯиЁҖпјҲData Query Language, DQLпјү еҹәжң¬з»“жһ„жҳҜз”ұSELECTеӯҗеҸҘпјҢFROMеӯҗеҸҘпјҢWHEREеӯҗеҸҘз»„жҲҗзҡ„жҹҘиҜўеқ—гҖӮ

                                    • ж•°жҚ®ж“ҚзәөиҜӯиЁҖпјҲData Manipulation Language, DMLпјүжҳҜSQLиҜӯиЁҖдёӯпјҢиҙҹиҙЈеҜ№ж•°жҚ®еә“еҜ№иұЎиҝҗиЎҢж•°жҚ®и®ҝй—®е·ҘдҪңзҡ„жҢҮд»ӨйӣҶпјҢ д»ҘINSERTгҖҒUPDATEгҖҒDELETEдёүз§ҚжҢҮд»Өдёәж ёеҝғпјҢеҲҶеҲ«д»ЈиЎЁжҸ’е…ҘгҖҒжӣҙж–°дёҺеҲ йҷӨпјҢжҳҜејҖеҸ‘д»Ҙж•°жҚ®дёәдёӯеҝғзҡ„еә”з”ЁзЁӢеәҸеҝ…е®ҡдјҡдҪҝз”ЁеҲ°зҡ„жҢҮд»ӨгҖӮ

                                    • ж•°жҚ®еә“е®ҡд№үиҜӯиЁҖDDL(Data Definition Language)пјҢжҳҜз”ЁдәҺжҸҸиҝ°ж•°жҚ®еә“дёӯиҰҒеӯҳеӮЁзҡ„зҺ°е®һдё–з•Ңе®һдҪ“зҡ„иҜӯиЁҖгҖӮ

                                    • DCLпјҲData Control LanguageпјүжҳҜж•°жҚ®еә“жҺ§еҲ¶иҜӯиЁҖгҖӮ жҳҜз”ЁжқҘи®ҫзҪ®жҲ–жӣҙж”№ж•°жҚ®еә“з”ЁжҲ·жҲ–и§’иүІжқғйҷҗзҡ„иҜӯеҸҘ еҢ…жӢ¬пјҲgrant,deny,revokeзӯүпјүиҜӯеҸҘгҖӮеңЁй»ҳи®ӨзҠ¶жҖҒдёӢпјҢеҸӘжңүsysadmin,dbcreator,db_ownerжҲ–db_securityadminзӯүдәәе‘ҳжүҚжңүжқғеҠӣжү§иЎҢDCLгҖӮ

                                    жҲ‘д»¬е№іж—¶жүҖиҜҙзҡ„CRUDе…¶е®һе°ұжҳҜеўһеҲ ж”№жҹҘпјҲCreate/Retrieve/Update/Deleteпјү

                                    ж•°жҚ®еә“е®ҡд№үиҜӯиЁҖпјҲDDLпјү

                                    SQLж•°жҚ®зұ»еһӢ

                                    д»ҘдёӢзҡ„ж•°жҚ®зұ»еһӢз”ЁдәҺеӯ—з¬ҰдёІеӯҳеӮЁпјҡ

                                    • char(n)еҸҜд»ҘеӯҳеӮЁд»»ж„Ҹеӯ—з¬ҰдёІпјҢдҪҶжҳҜжҳҜеӣәе®ҡй•ҝеәҰдёәnпјҢеҰӮжһңжҸ’е…Ҙзҡ„й•ҝеәҰе°ҸдәҺе®ҡд№үй•ҝеәҰж—¶пјҢеҲҷз”Ёз©әж јеЎ«е……гҖӮ
                                    • varchar(n)д№ҹеҸҜд»ҘеӯҳеӮЁд»»ж„Ҹж•°йҮҸеӯ—з¬ҰдёІпјҢй•ҝеәҰдёҚеӣәе®ҡпјҢдҪҶдёҚиғҪи¶…иҝҮnпјҢдёҚдјҡз”Ёз©әж јеЎ«е……гҖӮ

                                    д»ҘдёӢж•°жҚ®зұ»еһӢз”ЁдәҺеӯҳеӮЁж•°еӯ—пјҡ

                                    • smallintз”ЁдәҺеӯҳеӮЁе°Ҹзҡ„ж•ҙж•°пјҢиҢғеӣҙеңЁ (-32768пјҢ32767)
                                    • intз”ЁдәҺеӯҳеӮЁдёҖиҲ¬зҡ„ж•ҙж•°пјҢиҢғеӣҙеңЁ (-2147483648пјҢ2147483647)
                                    • bigintз”ЁдәҺеӯҳеӮЁеӨ§еһӢж•ҙж•°пјҢиҢғеӣҙеңЁ (-9,223,372,036,854,775,808пјҢ9,223,372,036,854,775,807)
                                    • floatз”ЁдәҺеӯҳеӮЁеҚ•зІҫеәҰе°Ҹж•°
                                    • doubleз”ЁдәҺеӯҳеӮЁеҸҢзІҫеәҰзҡ„е°Ҹж•°

                                    д»ҘдёӢж•°жҚ®зұ»еһӢз”ЁдәҺеӯҳеӮЁж—¶й—ҙпјҡ

                                    • dateеӯҳеӮЁж—Ҙжңҹ
                                    • timeеӯҳеӮЁж—¶й—ҙ
                                    • yearеӯҳеӮЁе№ҙд»Ҫ
                                    • datetimeз”ЁдәҺж··еҗҲеӯҳеӮЁж—Ҙжңҹ+ж—¶й—ҙ

                                    еҲ—зә§зәҰжқҹжқЎд»¶

                                    еҲ—зә§зәҰжқҹжңүе…ӯз§Қпјҡ

                                    • дё»й”® Primary key зЎ®дҝқеҲ—дёӯзҡ„жҜҸдёӘеҖјйғҪжҳҜе”ҜдёҖзҡ„пјҢе№¶дё”дёҚиғҪдёәз©әгҖӮдёҖдёӘиЎЁеҸӘиғҪжңүдёҖдёӘдё»й”®пјҢдё»й”®еҲ—зҡ„еҖјз”ЁжқҘе”ҜдёҖж ҮиҜҶжҜҸдёҖиЎҢж•°жҚ®гҖӮеңЁдё»й”®еҲ—дёӯпјҢдёҚе…Ғи®ёжңүйҮҚеӨҚзҡ„еҖје’Ңз©әеҖјгҖӮ

                                      id INT PRIMARY KEY иЎЁзӨә id еҲ—жҳҜдё»й”®пјҢдё”дёҚиғҪдёәз©әдё”е”ҜдёҖгҖӮ

                                    • еӨ–й”® foreign key з”ЁдәҺе»әз«Ӣе’Ңз»ҙжҠӨдёӨиЎЁд№Ӣй—ҙзҡ„е…ізі»пјҢзЎ®дҝқеҲ—дёӯзҡ„еҖјеҝ…йЎ»еӯҳеңЁдәҺеҸҰдёҖиЎЁзҡ„дё»й”®жҲ–е”ҜдёҖзәҰжқҹеҲ—дёӯгҖӮе®ғдҝқиҜҒж•°жҚ®зҡ„еҸӮз…§е®Ңж•ҙжҖ§гҖӮеҚіеӨ–й”®еҲ—дёӯзҡ„ж•°жҚ®еҖјеҝ…йЎ»еҜ№еә”дәҺеҸҰдёҖиЎЁзҡ„дё»й”®жҲ–е”ҜдёҖеҲ—дёӯзҡ„жҹҗдёӘеҖјгҖӮ

                                      FOREIGN KEY (department_id) REFERENCES departments(id)пјҢиЎЁзӨә department_id еҲ—жҳҜеӨ–й”®пјҢеҸӮз…§ departments иЎЁзҡ„ id еҲ—гҖӮ

                                    • е”ҜдёҖ unique зЎ®дҝқеҲ—дёӯзҡ„жҜҸдёӘеҖјйғҪжҳҜе”ҜдёҖзҡ„пјҢдҪҶе…Ғи®ёз©әеҖјгҖӮдёҺдё»й”®зұ»дјјпјҢе”ҜдёҖзәҰжқҹдҝқиҜҒеҲ—дёӯжүҖжңүзҡ„йқһз©әж•°жҚ®йғҪжҳҜе”ҜдёҖзҡ„гҖӮдёҖдёӘиЎЁеҸҜд»ҘжңүеӨҡдёӘе”ҜдёҖзәҰжқҹпјҢдёҚеҗҢдәҺдё»й”®пјҢе”ҜдёҖзәҰжқҹе…Ғи®ёжңүз©әеҖјгҖӮ

                                      email VARCHAR(255) UNIQUE иЎЁзӨә email еҲ—зҡ„еҖјжҳҜе”ҜдёҖзҡ„гҖӮ

                                    • жЈҖжҹҘ check пјҲMySQLдёҚж”ҜжҢҒпјү з”ЁдәҺе®ҡд№үеҲ—зҡ„еҖјеҝ…йЎ»ж»Ўи¶іжҹҗдәӣжқЎд»¶гҖӮжЈҖжҹҘзәҰжқҹйҖҡеёёз”ЁдәҺйҷҗеҲ¶жҹҗдәӣзү№е®ҡзҡ„ж•°еҖјиҢғеӣҙжҲ–жЁЎејҸгҖӮ

                                      age INT CHECK (age >= 18)пјҢиЎЁзӨә age еҲ—зҡ„еҖјеҝ…йЎ»еӨ§дәҺжҲ–зӯүдәҺ 18гҖӮ

                                    • й»ҳи®Ө default з”ЁдәҺдёәеҲ—жҢҮе®ҡй»ҳи®ӨеҖјгҖӮеҰӮжһңеңЁжҸ’е…Ҙж•°жҚ®ж—¶жІЎжңүдёәиҜҘеҲ—жҸҗдҫӣеҖјпјҢеҲҷдјҡдҪҝз”Ёй»ҳи®ӨеҖјгҖӮй»ҳи®ӨеҖјд»…еңЁжҸ’е…Ҙж•°жҚ®ж—¶дҪҝз”ЁпјҢеҰӮжһңжҸ’е…Ҙж—¶иҜҘеҲ—жңүжҳҺзЎ®еҖјпјҢеҲҷй»ҳи®ӨеҖјдёҚдјҡз”ҹж•ҲгҖӮ

                                      status VARCHAR(10) DEFAULT 'active'пјҢиЎЁзӨә status еҲ—еңЁжңӘжҢҮе®ҡеҖјж—¶й»ҳи®ӨеҖјдёә 'active'гҖӮ

                                    • йқһз©ә/з©әеҖј not null/ null з”ЁдәҺе®ҡд№үеҲ—жҳҜеҗҰеҸҜд»Ҙдёәз©әгҖӮNOT NULL зәҰжқҹиҰҒжұӮиҜҘеҲ—зҡ„жҜҸдёҖиЎҢеҝ…йЎ»жңүдёҖдёӘйқһз©әеҖјпјҢиҖҢ NULL е…Ғи®ёеҲ—дёӯзҡ„жҹҗдәӣиЎҢжІЎжңүеҖјгҖӮ

                                      name VARCHAR(100) NOT NULL иЎЁзӨә name еҲ—дёҚиғҪдёә NULLпјҢжҜҸдёҖиЎҢеҝ…йЎ»жңүеҖјгҖӮ

                                    иЎЁзә§зәҰжқҹжқЎд»¶

                                    иЎЁзә§зәҰжқҹжңүеӣӣз§Қпјҡдё»й”®гҖҒеӨ–й”®гҖҒе”ҜдёҖгҖҒжЈҖжҹҘ

                                    • дё»й”® дё»й”®зәҰжқҹдҝқиҜҒиЎЁдёӯзҡ„жҜҸдёҖиЎҢж•°жҚ®йғҪжңүе”ҜдёҖзҡ„ж ҮиҜҶз¬ҰпјҢдё”иҜҘеҲ—зҡ„еҖјдёҚиғҪдёәз©әгҖӮдёҖдёӘиЎЁеҸӘиғҪжңүдёҖдёӘдё»й”®гҖӮ

                                      CONSTRAINT pk_primary_key PRIMARY KEY (id)

                                    • еӨ–й”® еӨ–й”®зәҰжқҹз”ЁдәҺеңЁдёӨдёӘиЎЁд№Ӣй—ҙе»әз«ӢеҸӮз…§е®Ңж•ҙжҖ§е…ізі»пјҢзЎ®дҝқжҹҗеҲ—зҡ„еҖјеҝ…йЎ»еӯҳеңЁдәҺеҸҰдёҖиЎЁзҡ„дё»й”®жҲ–е”ҜдёҖзәҰжқҹеҲ—дёӯгҖӮ

                                    CREATE TABLE Departments (
    DepartmentID INT,
    DepartmentName VARCHAR(50) NOT NULL,
    Location VARCHAR(50),
    ManagerID INT,
    Budget DECIMAL(10, 2),

    CONSTRAINT PK_Departments PRIMARY KEY (DepartmentID),
    CONSTRAINT UC_DepartmentName UNIQUE (DepartmentName),
    CONSTRAINT CK_BudgetPositive CHECK (Budget >= 0),
    CONSTRAINT FK_Manager FOREIGN KEY (ManagerID) REFERENCES Employees(EmployeeID),
    CONSTRAINT DF_Location DEFAULT 'Headquarters' FOR Location
);

                                    жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮADDжқҘж·»еҠ дёҖдёӘж–°зҡ„еҲ—пјҢйҖҡиҝҮDROPжқҘеҲ йҷӨдёҖдёӘеҲ—пјҢдёҚиҝҮжҲ‘д»¬еҸҜд»Ҙж·»еҠ restrictжҲ–cascadeпјҢй»ҳи®ӨжҳҜrestrictпјҢиЎЁзӨәеҰӮжһңжӯӨеҲ—дҪңдёәе…¶д»–иЎЁзҡ„зәҰжқҹжҲ–и§Ҷеӣҫеј•з”ЁеҲ°жӯӨеҲ—ж—¶пјҢе°Ҷж— жі•еҲ йҷӨпјҢиҖҢcascadeдјҡејәеҲ¶иҝһеёҰеј•з”ЁжӯӨеҲ—зҡ„зәҰжқҹгҖҒи§ҶеӣҫдёҖиө·еҲ йҷӨгҖӮиҝҳеҸҜд»ҘйҖҡиҝҮALTERжқҘдҝ®ж”№жӯӨеҲ—зҡ„еұһжҖ§гҖӮ

                                    еҲ йҷӨиЎЁ DROP

                                    жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮdrop tableжқҘеҲ йҷӨдёҖдёӘиЎЁпјҡ

                                    DROP TABLE иЎЁеҗҚ[restrict|cascade]

                                    е…¶дёӯrestrictе’ҢcascadeдёҠйқўзҡ„ж•ҲжһңдёҖиҮҙгҖӮ

                                    ж•°жҚ®еә“ж“ҚзәөиҜӯиЁҖ (DML)

                                    жҸ’е…Ҙж•°жҚ® INSERT INTO

                                    йҖҡиҝҮдҪҝз”Ёinsert intoиҜӯеҸҘжқҘеҗ‘ж•°жҚ®еә“дёӯжҸ’е…ҘдёҖжқЎж•°жҚ®пјҲдёҖжқЎи®°еҪ•пјүпјҡ

                                    INSERT INTO иЎЁеҗҚ VALUES(еҖј1, еҖј2, еҖј3)

                                    еҰӮжһңжҸ’е…Ҙзҡ„ж•°жҚ®дёҺеҲ—дёҖдёҖеҜ№еә”пјҢйӮЈд№ҲеҸҜд»ҘзңҒз•ҘеҲ—еҗҚпјҢдҪҶжҳҜеҰӮжһңеёҢжңӣеҗ‘жҢҮе®ҡеҲ—дёҠжҸ’е…Ҙж•°жҚ®пјҢе°ұйңҖиҰҒз»ҷеҮәеҲ—еҗҚпјҡ

                                    INSERT INTO иЎЁеҗҚ(еҲ—еҗҚ1, еҲ—еҗҚ2) VALUES(еҖј1, еҖј2)

                                    INSERT INTO иЎЁеҗҚ(еҲ—еҗҚ1, еҲ—еҗҚ2) VALUES(еҖј1, еҖј2), (еҖј1, еҖј2), (еҖј1, еҖј2)

                                    жҲ‘д»¬жқҘиҜ•иҜ•зңӢеҗ‘жҲ‘д»¬еҲҡеҲҡеҲӣе»әзҡ„иЎЁдёӯж·»еҠ дёүжқЎж•°жҚ®гҖӮ

                                    дҝ®ж”№ж•°жҚ® UPDATE

                                    жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮupdateиҜӯеҸҘжқҘжӣҙж–°иЎЁдёӯзҡ„ж•°жҚ®пјҡ

                                    UPDATE иЎЁеҗҚ SET еҲ—еҗҚ=еҖј,... WHERE жқЎд»¶

                                    еҲ йҷӨж•°жҚ®
